rmport · 端口清理

一行命令杀掉占用指定端口的进程。替代 lsof → 找 PID → kill 三步操作。

$ npx rmport 3000
✓ Killed node (PID 12345) on port 3000.
$ npx rmport 3000

支持 macOS / Linux / Windows，Node.js ≥ 18
