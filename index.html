开发者工具箱

一个域名，多个能力。图片即接口，粘贴即使用。

https://boluo66.top/{工具}/?{参数}
🖼

favimg · 图标获取

一行 URL，一张图标。自动提取任意网站的 favicon。可选 ?size=64 指定尺寸。

GET /favimg/?url=xxx
📱

qr · 二维码

一行 URL，一张二维码。任意内容，扫码直达。

GET /qr/?content=xxx
📋

og · 分享卡片

一行 URL，一张分享卡片。自动提取标题和描述。

GET /og/?url=xxx
🔥

git-heat · 年度报告

Git 年度代码报告。在任意 Git 仓库下运行，自动生成精美的可视化年报。

$ cd 你的项目
$ npx git-heat

⚠️ 需要在本地 Git 仓库目录下运行
