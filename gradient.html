Gradient · 渐变生成

← 工具箱

📖 用法

一个 URL 返回一张渐变色 SVG 图片。改参数就能换颜色、尺寸、角度。 浏览器直接打开就能看到，放在 <img> 里当占位图也行。

参数

参数说明默认
from起始颜色#615ced
to结束颜色#00a884
w宽度（px）1920
h高度（px）1080
angle渐变角度135

🎨 示例

紫色到绿色

https://boluo66.top/gradient/?from=%23615ced&to=%2300a884&w=600&h=200

品牌色到橙

https://boluo66.top/gradient/?from=%23615ced&to=%23ff8a00&w=600&h=200

横向渐变桌面壁纸

https://boluo66.top/gradient/?from=%230d1117&to=%23615ced&w=1920&h=1080&angle=0

🖼 HTML 中使用

<img src="https://boluo66.top/gradient/?from=#615ced&to=#00a884&w=800&h=200" alt="背景">

🛠 CSS 中使用

background: url(https://boluo66.top/gradient/?from=#0d1117&to=#615ced&w=1920&h=600) center/cover;
